# NostalgiaRank AI

A casual AI-powered web game where you can rank ANYTHING - from movies to pizza toppings to sports cars. Just type what you want to rank, and AI generates a custom game instantly!

## Features - 100% FREE!

- **Rank Anything**: Generate custom ranking games for any topic you can think of
- **AI-Powered Lists**: Free Llama 3.2 AI creates 10 relevant items to rank
- **Unlimited Possibilities**: Movies, music, food, sports, games, and more
- **Smart Rankings**: AI provides consensus/popularity rankings for comparison
- **Completely Free**: Powered by Hugging Face's free inference API - no credit card needed!

### Core Features
- **Interactive Drag-and-Drop**: Intuitive ranking interface with smooth animations
- **Results Comparison**: See how your taste compares to consensus/actual data
- **Nostalgic Design**: Beautiful, playful UI with gradient backgrounds
- **Mobile Responsive**: Works seamlessly on desktop and mobile devices
- **No Login Required**: Instant play with no account needed

## Tech Stack

- **React** - UI components
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling
- **@dnd-kit** - Drag-and-drop functionality
- **Hugging Face Inference API** - FREE AI-powered game generation (Llama 3.2)
- **Vercel Serverless Functions** - Backend API

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm
- Hugging Face API token (FREE - for AI mode)

### Installation

```bash
# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Add your HUGGINGFACE_API_KEY to .env

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The app will be available at `http://localhost:5173/`

### Getting a FREE Hugging Face API Token

1. Go to [https://huggingface.co/](https://huggingface.co/)
2. Sign up (completely free, no credit card required!)
3. Navigate to Settings â†’ Access Tokens: [https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)
4. Click "New token"
5. Give it a name and select "Read" permissions
6. Copy your token (starts with `hf_`)
7. Add it to your `.env` file as `HUGGINGFACE_API_KEY=hf_your_token_here`

## How to Play

### Classic Mode
1. **Choose Classic Mode**: Click on the Cartoon Network option
2. **Rank Shows**: Drag and drop 10 classic shows from favorite to least favorite
3. **See Results**: Compare your ranking to actual viewership data
4. **Play Again**: Try ranking with a different mindset

### AI Mode
1. **Choose AI Mode**: Click on the AI-powered option
2. **Enter Your Prompt**: Type what you want to rank (e.g., "Best 90s action movies")
3. **AI Generates Game**: Wait 5-10 seconds while Claude creates your custom game
4. **Rank Items**: Drag and drop the AI-generated items
5. **See Results**: Compare your ranking to the AI's consensus ranking
6. **Try Another**: Create unlimited custom games!

## Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) and import your repository
3. Add environment variable:
   - Key: `HUGGINGFACE_API_KEY`
   - Value: Your Hugging Face token
4. Deploy!

The serverless function will automatically work on Vercel.

## Environment Variables

- `HUGGINGFACE_API_KEY` - Your FREE Hugging Face API token (required for AI mode)
